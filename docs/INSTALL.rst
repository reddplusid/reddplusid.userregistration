reddplusid.userregistration Installation
----------------------------------------

To install reddplusid.userregistration using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``reddplusid.userregistration`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        reddplusid.userregistration

* Re-run buildout, e.g. with:

    $ ./bin/buildout
